# RAG HR Policy Assistant

A professional RAG (Retrieval-Augmented Generation) system that allows users to upload HR policy PDFs and ask questions about them.

## Features

- PDF document upload and text extraction
- Context-aware question answering
- Source citations in responses
- Clean, professional UI
- Mobile responsive

## Tech Stack

- HTML5, CSS3, JavaScript
- Google Gemini API
- PDF.js for document processing

## Setup

1. Clone this repository
2. API key is already configured in `config.js`
3. Open `index.html` in a browser or host on GitHub Pages

## How to Use

1. Upload an HR policy PDF document
2. Wait for text extraction to complete
3. Ask questions about the policy
4. Get accurate, sourced answers

## Live Demo

Host on GitHub Pages:
1. Go to repository Settings
2. Pages section
3. Select main branch
4. Save
